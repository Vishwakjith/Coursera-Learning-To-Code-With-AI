# Understanding Firebase

## Realtime Database Setup

Create a application where we,
Setup a Realtime Database of Firebase and send data to the DB.
Whenever data is published to DB, the UI should automatically be upto-date with data present in DB.

## Like & Dislike

For every message, add a like and dislike button.
Along with it show a totale count of likes and dislikes for that respective message.
Make sure it is real-time updates.

## Delete All

Add a button to delete all the messages that were sent.
